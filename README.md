# 🚑 AstanaEMS — AI-диспетчер скорой помощи

> SmartScape Hackathon 2026 | Track 3: City Safety & Social Services | STEMNOVA.kz

## Проблема

В Астане из 145 машин скорой на линию выходят лишь 63. Диспетчеры распределяют бригады вручную, без учёта того, где вызовы появятся в ближайший час. Каждая лишняя минута ожидания при остановке сердца снижает шанс выжить на 6–10%.

## Решение

AstanaEMS — прототип AI-системы, которая предсказывает количество вызовов скорой по каждому району Астаны на следующий час и рекомендует диспетчеру, куда заранее переместить свободные машины.

Модель учитывает время суток, день недели, погоду, температуру, гололёд, праздники и крупные городские события.

## Как запустить

### ⚡ Быстрый запуск (Windows)

1. Клонировать репозиторий:
```bash
git clone https://github.com/Volnikus/SmartScapeAIHackaton.git
```
2. Открыть папку и дважды кликнуть **`run_app.bat`**

Лаунчер автоматически:
- найдёт Python на компьютере
- проверит наличие всех библиотек
- установит недостающие из `requirements.txt`
- откроет приложение в браузере

> Требуется Python 3.10+ с официального сайта [python.org](https://www.python.org/downloads/). При установке обязательно включить **"Add Python to PATH"**.

### 🖥️ Ручной запуск

```bash
git clone https://github.com/Volnikus/SmartScapeAIHackaton.git
cd SmartScapeAIHackaton
pip install -r requirements.txt
streamlit run app.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`

## Как работает модель

1. Генерируется синтетический датасет из 50 000 строк с реалистичными паттернами Астаны
2. LightGBM обучается на исторических данных вызовов с учётом погоды, времени и районов
3. Модель предсказывает количество вызовов по каждому из 5 районов на следующий час
4. Streamlit-интерфейс отображает тепловую карту риска и рекомендации по расстановке машин
